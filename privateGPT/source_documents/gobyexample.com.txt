[https://gobyexample.com/]Go by Example

   [https://go.dev/]Go is an open source programming language designed for
   building simple, fast, and reliable software. Please read the official
   [https://go.dev/doc/tutorial/getting-started]documentation to learn a
   bit about Go code, tools packages, and modules.
