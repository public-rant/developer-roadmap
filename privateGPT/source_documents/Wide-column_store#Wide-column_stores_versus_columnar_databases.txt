   From Wikipedia, the free encyclopedia

   A wide-column store (or extensible record store) is a
   [https://en.wikipedia.org/wiki/Column-oriented_DBMS]column-oriented
   DBMS and therefore a special type of
   [https://en.wikipedia.org/wiki/NoSQL]NoSQL
   [https://en.wikipedia.org/wiki/Database]database.^[https://en.wikipedia
   .org/wiki/Wide-column_store#cite_note-db-engines-1][1] It uses tables,
   rows, and columns, but unlike a
   [https://en.wikipedia.org/wiki/Relational_database]relational database,
   the names and format of the columns can vary from row to row in the
   same table. A wide-column store can be interpreted as a two-dimensional
   [https://en.wikipedia.org/wiki/Key%E2%80%93value_store]key–value
   store.^[https://en.wikipedia.org/wiki/Wide-column_store#cite_note-db-en
   gines-1][1]

Wide-column stores versus columnar
databases[[https://en.wikipedia.org/w/index.php?title=Wide-column_store&actio
n=edit&section=1]edit]

   Wide-column stores such as
   [https://en.wikipedia.org/wiki/Bigtable]Bigtable and
   [https://en.wikipedia.org/wiki/Apache_Cassandra]Apache Cassandra are
   not [https://en.wikipedia.org/wiki/Column-oriented_DBMS]column stores
   in the original sense of the term, since their two-level structures do
   not use a columnar data layout. In genuine column stores, a columnar
   data layout is adopted such that each column is stored separately on
   disk. Wide-column stores do often support the notion of
   [https://en.wikipedia.org/wiki/Column_family]column families that are
   stored separately. However, each such column family typically contains
   multiple columns that are used together, similar to traditional
   relational database tables. Within a given column family, all data is
   stored in a row-by-row fashion, such that the columns for a given row
   are stored together, rather than each column being stored separately.

   Wide-column stores that support column families are also known as
   column family
   databases.^[[https://en.wikipedia.org/wiki/Wikipedia:Citation_needed]ci
   tation needed]
