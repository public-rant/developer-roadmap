In this article

   Applies to: [yes-icon.svg?view=sql-server-ver15]
   [https://learn.microsoft.com/en-us/sql/sql-server/sql-docs-navigation-g
   uide?view=sql-server-ver15#applies-to]SQL Server

   SQL Server provides tutorials to help you learn new technologies and
   features. Tutorials for earlier versions can usually be used with more
   recent versions. When tutorials address a newer feature, they will
   indicate the minimum version to complete the tutorial.
     * If a tutorial uses one of the Adventure Works databases, you should
       download and use the most recent version from CodePlex. For more
       information, see
       [https://github.com/microsoft/sql-server-samples/releases/tag/adven
       tureworks]AdventureWorks Databases and Scripts for SQL Server 2016
       CTP3.
     * Download a Power Pivot workbook and Power View reports to use with
       Power View tutorials, go to
       [https://go.microsoft.com/fwlink/?LinkId=220734]Power View Samples
       for SQL Server 2012.
       Note
       Power View support is no longer available after SQL Server 2017.
     * Check out the new
       [https://learn.microsoft.com/en-us/sql/samples/wide-world-importers
       -what-is?view=sql-server-ver15]Wide World Importers sample
       available on the
       [https://github.com/Microsoft/sql-server-samples]sql-server-samples
       GitHub repository.

Current tutorials

   Technology Description
   [https://learn.microsoft.com/en-us/analysis-services/analysis-services-
   tutorials-ssas]Analysis Services Tutorials (SSAS) Learn how to develop
   and deploy Analysis Services tabular or multidimensional models. Also
   learn how to use tools such as PowerPivot to analyze the data in the
   models.
   [https://learn.microsoft.com/en-us/sql/relational-databases/database-en
   gine-tutorials?view=sql-server-ver15]Database Engine Tutorials Learn
   how to connect to an instance of the database engine and then build and
   run Transact-SQL statements.
   [https://wiki.scn.sap.com/wiki/display/EIM/EIM+Tutorials]Enterprise
   Information Management Tutorials Learn how to manage information in an
   enterprise using Microsoft EIM technologies.
   [https://learn.microsoft.com/en-us/sql/integration-services/integration
   -services-tutorials?view=sql-server-ver15]Integration Services
   Tutorials Learn how to create and deploy SSIS packages.
   [https://learn.microsoft.com/en-us/sql/relational-databases/replication
   /replication-tutorials?view=sql-server-ver15]Replication Tutorials
   Learn how to set up and run replication topologies using SQL Server
   Management Studio.
   [https://learn.microsoft.com/en-us/sql/reporting-services/reporting-ser
   vices-tutorials-ssrs?view=sql-server-ver15]Reporting Services Tutorials
   (SSRS) Learn how to create basic reports and subscriptions.
   [https://learn.microsoft.com/en-us/sql/machine-learning/tutorials/r-tut
   orials?view=sql-server-ver15]SQL Server Machine Learning Tutorials
   Learn how to work with SQL Server data using R or Python. Run R or
   Python scripts from Transact-SQL.
